Sorting types: natural, date

In this example, different sort types have been applied.

First in the "BirthDate" column: the "date" sort type is used, handy for the uk format: dd/mm/yy.

In both "Mail" and "Salary" columns, the "natural" sort type is used. Internally it uses the naturalSort() function by Jim Palmer to provide natural sorting in DataTables.
