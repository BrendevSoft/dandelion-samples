Interacting with the asset graph

Using the Thymeleaf dialect

In this example, a bundle is activated using the Thymeleaf dialect: 

<body ddl:bundle-includes="bundle-using-the-dialect">

As a consequence, the following script is injected in the HTML source code: 

<script src="/dandelion-thymeleaf-starter/assets/js/asset-dialect.js"></script>
